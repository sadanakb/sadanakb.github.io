# sadanakb.github.io

Mein persoenliches Portfolio — gebaut mit Astro, Tailwind CSS.

**Live:** [sadanakb.github.io](https://sadanakb.github.io)

## Features

- Bilingual (Deutsch / English) mit automatischer Locale-Erkennung
- Dark-Mode Design mit Gradient-Mesh und Scroll-Reveal-Animationen
- 5 Projekt-Showcases mit Screenshots und Feature-Listen
- SEO-optimiert: JSON-LD, Open Graph, Sitemap, Canonical URLs
- Automatisches Deployment via GitHub Actions

## Tech Stack

- [Astro](https://astro.build) — Static Site Generator
- [Tailwind CSS](https://tailwindcss.com) — Styling
- GitHub Pages — Hosting
- GitHub Actions — CI/CD

## Lokal starten

```bash
npm install
npm run dev
```

Oeffnet `http://localhost:4321`.
